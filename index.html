⚙ Integração com Google Sheets

📋 Como configurar o Google Apps Script

  1. Acesse script.google.com e crie um novo projeto
  2. Cole o código abaixo no editor e salve (Ctrl+S)
  3. Clique em Implantar → Nova implantação
  4. Selecione tipo: App Web
  5. Em "Executar como": Eu; em "Quem tem acesso": Qualquer pessoa
  6. Clique em Implantar e copie a URL gerada
  7. Cole a URL no campo abaixo e clique em Salvar

💡 Nota: Os dados também são salvos localmente no navegador (localStorage). A integração com Google Sheets é opcional mas recomendada para persistência.
